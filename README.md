## 簡介

這是一個展示 Markdown 各種格式用法的範例文件。

---

## 功能特色

- **粗體文字**：用於強調重要內容
- *斜體文字*：用於標示專有名詞或書名
- ~~刪除線~~：用於表示已過時或移除的內容
- `行內程式碼`：用於標示程式碼片段
